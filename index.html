SA-QA E-Voting Website

The e-voting system was developed to support digital, structured, and secure election voting and vote counting processes. The system aims to improve efficiency, accuracy, and transparency in elections while ensuring data validation, system reliability, and ease of use for both voters and election administrators.

Role: System Analyst – Quality Assurance (SA–QA)
- Analyzed system requirements based on business processes and election regulations.
- Prepared the Software Requirement Specification (SRS) as a reference for system development.
- Designed use cases, flowcharts, and test scenarios to ensure the system meets requirements.
- Performed functional testing on core e-voting features such as voter authentication, voting process, and vote recapitulation.
- Identified, documented, and reported bugs and system inconsistencies to the development team.
- Ensured the system met usability, reliability, and data consistency standards before implementation.

E-Posyandu Website – UI/UX Design

The E-Posyandu website interface was designed to support the digitalization of maternal and child health services through efficient health data management. The website aims to assist community health volunteers (kader) and healthcare staff in accessing, recording, and monitoring information easily through a simple, informative, and user-friendly interface.

Role: Web Designer (UI/UX)
- Designed the E-Posyandu website interface with a strong focus on usability for health volunteers and medical staff.
